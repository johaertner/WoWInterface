Be courteous of other players while looking for the treasures, a little kindness goes a long way!

! IMPORTANT !

This addon is powered by TomTom, and you MUST have it installed in order to run LostTreasures!
If you do not have TomTom installed, you can download it here.
Note: If you download this addon from Curse, TomTom should be included in the download!

LostTreasures may interfere with other addons due to portions of TomTom being modified to achieve functionality.

Features

  • Marks treasures of Pandaria on the map giving you an in-game guide!
  • Provides helpful notes on where treasures are located and how to get there.
  • Many options to make the addon work the way you want it to!
    • See images for a look at the available options.
  • An easy to use settings UI. Slash command: /lt
    • Game Options (Esc) > Interface > AddOns > Lost Treasures
